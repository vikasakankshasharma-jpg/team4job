# **App Name**: CCTV Job Connect

## Core Features:

- User Registration & Authentication: Secure user registration and authentication using email-based OTP verification for Job Givers and Installers, including Aadhar Card verification for installers. Login via mobile number or email.
- Job Posting & Management: Job Givers can easily post detailed job listings with location, budget, and bidding deadlines.
- Bidding System: Verified installers bid on jobs, ask questions, and participate in discussions via the job comment system.
- Installer Selection & Confirmation: Job Givers select up to 3 installers (or all if fewer). The first to confirm gets the job; funds are held in escrow pending Job Giver reconfirmation.
- AI Job Description Tool: AI generates compelling job descriptions from titles, ensuring clarity and attracting qualified installers. This is an AI tool.
- AI Bid Assistant: AI assists Installers in crafting effective bids. Junior installers can analyze job requirements and prepare competitive bids with an AI Chat assistant. This is an AI tool.
- Reputation and Tier System: A reputation system tracks points and awards tier badges based on completed jobs and ratings, promoting trust and recognizing excellence.
- Location System: Accurate location services using a PINCODE system with a detailed list of post offices, defaulting to the post office location on a map, with pinpoint accuracy adjustments.
- Role Based System: A streamlined role-based system differentiates between Job Givers and Installers, enabling seamless role switching without a complete logout.
- Desktop/Laptop Usability: The application is designed for seamless usability on desktop and laptop computers; also accessible as a web app.
- Google Platform Integration: Full utilization of Google platform tools and systems, guaranteeing live application functionality and broad compatibility within the Google ecosystem.
- Push Notifications: Instant real-time updates via push notifications for new job postings, bid statuses, and critical account activities.
- Secure Payment Gateway Integration: Integrated secure payment gateways to facilitate escrow payments, ensuring reliable financial transactions.
- Dispute Resolution System: A dedicated system to efficiently resolve disputes between Job Givers and Installers, ensuring fair outcomes.
- Job Comment System: Job-specific comments enable installers to ask questions before bidding, enhancing transparency and saving time.
- Time Remaining Display: Clearly displays remaining time (days and hours) for active bids, confirmation deadlines, and upcoming bid opportunities.
- Subscription Plans and Platform Charges: Clear structure for platform charges, including options for quarterly and yearly subscriptions, and bid bundles with one-year validity.
- Admin Portal: A fully controlled Admin portal with an organized dashboard for efficient application management.
- AI-Powered Smart Matching: Intelligent AI matches Job Givers with suitable installers based on skills, location, availability, and ratings using a tool for predictive analysis.
- Job Location Filtering: Advanced job filtering enables installers to define a preferred work area by location, radius, or PINCODES to organize bidding efforts.
- Review and Rating System Enhancement: Enhanced review system allows detailed feedback on installer skills and provides installers with ability to respond to the given reviews.
- Escrow Milestone Payments: Larger jobs can be divided into milestones, enabling Job Givers to release payments incrementally upon successful completion of each milestone.
- Reporting System: Comprehensive reporting system with data export for analyzing platform activity, user behavior, and finances, tailored for all roles.
- Installer Skill Verification: Installers can upload certifications and licenses, which the platform can verify, increasing trust.

## Style Guidelines:

- Primary color: Desaturated grayish-blue (#B0B6C4) to create a professional feel.
- Background color: Very light grayish-blue (#F0F2F5), a lighter variation of the primary to create a clean backdrop.
- Accent color: Muted amber (#C88E4D), analogous to primary to bring key CTAs, but with desaturation.
- Font: 'Inter' (sans-serif) for excellent readability and a modern feel.
- Simple, clear icons that align with the professional aesthetic within job cards and navigation.
- Clean and modern layout with generous whitespace. Mobile navigation at the bottom for easy access.
- Subtle transitions and animations to enhance user experience, like loading indicators and smooth modal transitions.
- Implement a consistent grid system to ensure proper alignment and spacing of elements.
